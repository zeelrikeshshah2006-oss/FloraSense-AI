# 🌿 FloraSense AI

FloraSense AI is an intelligent flower species recognition and disease detection platform. Powered by Deep Learning and Transfer Learning techniques, this application helps users quickly identify different types of flowers and detect potential plant diseases from images, making plant care easier, smarter, and more accessible.

---

## 🚀 Features

* **🌸 Flower Classification:** Automatically detect and classify various flower species using deep learning models.
* **🦠 Disease Detection:** Analyze flower leaves and petals to diagnose plant diseases instantly.
* **📜 History Tracking:** Keep a secure record of all your previous scan reports for progress monitoring.
* **🔒 Secure User Authentication:** A dedicated dashboard system with secure login and profile management.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Poppins Typography), Responsive UI
* **Backend:** Python, Flask
* **Machine Learning:** TensorFlow / Keras (`train_model.keras`)
* **Deep Learning Strategy:** Transfer Learning
